# Appsec-SDLC-Writeups

This repository contains short, developer-focused application security write-ups analyzing common web vulnerabilities from a software development lifecycle (SDLC) perspective.

The write-ups are based on controlled lab environments and focus on understanding root cause, secure design, and where issues should have been prevented or detected in the SDLC.
